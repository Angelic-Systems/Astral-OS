# Astral-OS

IMPORTANT! Astral-OS is a work in progress and is not yet ready for daily use! 

Astral OS is built for terminal power users. It uses the sway compositor/WM to render a full screen terminal emulator running the Zellij multiplexer to provide a tiling WM like expereance that doesn't break your flow.

comes pre-installed with the following applications:

plymouth,
sway,
zellij,
neovim,
nano,
zjstatus,
kitty,
btop,
yazi,
wiremix,
bluetui,
impala,

keybinds:

super + K = keybinding guide 
super + return = open terminal window,
super + W = close window,
super + shift + b = lynx browser,
