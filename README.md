# Astral-OS

IMPORTANT! Astral-OS is a work in progress and is not yet ready for daily use! 

comes pre-installed with the following applications:

plymouth,
sway,
zellij,
neovim,
nano,
zjstatus,
kitty,
btop,
yazi,
wiremix,
bluetui,
impala,

keybinds:

super + K = keybinding guide 
super + return = open terminal window,
super + W = close window,
super + shift + b = lynx browser,
